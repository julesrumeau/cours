Les deux type de balises

Il existe deux types de balises :

  1. les balises usuelles encadrent du code, par exemple : <p> et </p>
  2. les balises dites orphelines n'encadrent aucun point ; elles se ferment elles-mêmes. Par exemple : <br/> abrévation de break row; <hr/> abrévation de horizontal row;
